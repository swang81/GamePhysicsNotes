
# 简介

《Game Physics and Development Notes》是一份围绕游戏物理与游戏开发实践的长期笔记，记录了从物理建模到工程实现，再到实际游戏应用过程中的关键思考与取舍。

本笔记面向具备一定技术背景、希望深入理解**游戏物理“为什么这样实现”**的开发者，强调原理、工程经验与实际问题之间的联系，而非公式堆砌或 API 使用说明。
